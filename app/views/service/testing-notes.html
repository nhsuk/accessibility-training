Testing notes

To help find accessibility issues you could use the WAVE browser extension, which is available for Chrome, Firefox or Microsoft Edge.

Please note that WAVE, as with other automated evaluation tools, will not find all the accessibility issues on a web page. They can only test some of the accessibility requirements that do not need human intervention. Manual testing by humans must be done to ensure an accessible user experience.

Testing checklist

  • Does WAVE find any errors?
  • Do page titles make sense?
  • Are headings used correctly?
  • Can you access everything using the keyboard?
  • Are form fields labeled correctly?
  • Are there any colour contrast errors or reliance on colour alone?

The NHS Digital service manual has further guidance on accessibility testing which you could refer to for more checks.
